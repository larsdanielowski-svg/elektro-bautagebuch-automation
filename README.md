# Elektrofirma Bautagebuch Automation - Proof of Concept

## Projektübersicht
Web-App für Elektrofirma zur automatischen Erstellung von Bautagebüchern aus Baustellenfotos.

## Phase 1: Proof of Concept
- Einfache Web-App mit Bildupload
- Mock KI-Analyse (später OpenAI Vision)
- Automatische Texterstellung
- Bautagebuch-Ansicht
- Dark Mode & Responsive Design

## Technologie-Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js mit Express
- File Upload: Multer
- Mock KI: Simulierte Bildanalyse
- Datenbank: JSON Dateien (für PoC)
